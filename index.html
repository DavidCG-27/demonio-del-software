Modo Práctica de Código

Este modo te presenta fragmentos de código con "Code Smells" que debes refactorizar aplicando patrones de diseño.

Carpeta de Ejemplos

Existe una carpeta llamada `examples` en el repositorio con ejercicios de prueba.

Para usarla, simplemente arrastra esa carpeta entera (o los archivos `.txt` que contiene) a la zona de carga.

Subir tus propios archivos

Si creas tus propios ejercicios, usa esta nomenclatura para los archivos:

  • • Problema: ej01.txt
  • • Solución: ej01_sol.txt

Modo Patrones UML

Un sistema de "Flashcards" para memorizar la estructura de las clases de los patrones GoF.

  1. La aplicación te mostrará el nombre de un patrón aleatorio (ej: Strategy).
  2. Toma un papel y dibuja el diagrama de clases correspondiente.
  3. Pulsa "Ver Solución" para revelar el diagrama correcto generado con Mermaid.js.
  4. Compara tu dibujo con la solución oficial.
